Tattoo Task Tracker

Technology used:

  • React.js
  • HTML
  • CSS
  • JavaScript
  • Node.js
  • Express
  • PostgreSQL

Brief description of project:

My sister is a tattoo artist who like many other tattoo artists, self manages her schedule. There are a lot of small details involved and it can often get overwhelming for her. I created Tattoo Task Tracker as a way to help my sister and other tattoo artists who need the same help. Tattoo Task Tracker helps tattoo artists stay organized by giving them the ability to create and view schedules as well as manage their client and tattoo information.

Project Tracker

Technology used:

  • React.js
  • HTML
  • CSS
  • JavaScript
  • Node.js
  • Express
  • PostgreSQL

Brief description of project:

The company I currently work for was struggling to find an application to manage projects that met their needs without costing an arm and a leg. Project Tracker was created as an alternative option for them. Project Tracker is an application targeted at companies to help them manage projects at a team level. It allows tracking of to-do lists for each project as well as a priority. It also provides the ability to assign tasks from the to-do list to employees of the company.
